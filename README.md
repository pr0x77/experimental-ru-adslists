## 🛠️ Особенности
- **DOMAIN-SUFFIX**: Полное покрытие поддоменов для 100% работы.
- **Auto-Cleanup**: Робот сам удаляет дубликаты и пустые строки.
- **Apple Update**: Список Apple обновляется еженедельно из официальных источников.
